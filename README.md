# Photo SS (Photo Slide Show :가제)

시작페이지용 크롬 확장프로그램


심플하고 아름다운 시작페이지를 만듭니다.

<br/>

## 개발 기간

2017.11 - 개발중

<br/>
<br/>

## 프로젝트 설명

심플한 검색창과 북마크 기능(구현 미정)을 제공하고, 중점인 사진을 보여주는데 집중하는 웹.

사진 슬라이드를 배경으로 표시하거나 액자 형식으로 표시하는 등 사진 슬라이드의 표시 방식, 테마에 집중해서 개발.

<br/>
<br/>

## 개발 스택

- HTML5 / CSS3
- Sass (Scss)
- JavaScript (ES6)
- React (create-react-app)
